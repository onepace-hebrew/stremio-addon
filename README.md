# One Pace Hebrew Subtitles

Hebrew subtitles for One Pace (the fan-made recut of One Piece) — as a Stremio addon.

> ⚠️ These Hebrew subtitles are AI-generated and may contain errors.

## Install

1. Open **Stremio** → **Add-ons**.
2. Paste this URL into the **"Add-on Repository URL"** box and click **Install**:

   ```
   https://onepace-hebrew.github.io/stremio-addon/manifest.json
   ```

3. Play a One Pace episode (you need the One Pace stream addon installed) and pick the **Hebrew** subtitle track.

The plain track works in Stremio's player; for fancier styling, use [mpv](https://mpv.io/).

## Credits

Based on the English subtitles from the [One Pace project](https://onepace.net). Hebrew translation by **That One Gerbil**.
