Install

Add to Stremio

Copy the add-on repository URL below, or install it directly in one click.

https://stremio-addon.onepace-hebrew.workers.dev/manifest.json
Install in Stremio
  1. Add the URL above in Stremio.
  2. Play a One Pace episode (you need the official One Pace addon installed — see below).
  3. Pick the Hebrew subtitle track.

The plain track works in the Stremio player. For fancier styling, use mpv.

Fully compatible

Works with the official One Pace addon

This add-on provides subtitles only. Pair it with the official One Pace addon for Stremio, which provides the episode streams — install both and they work together out of the box.

Install ב‎Stremio
  1. פתחו את Stremio ועברו אל מסך התוספים Add-ons.
  2. הדביקו את הכתובת שלמעלה בתיבה "Add-on Repository URL" ולחצו על Install (התקנה).
  3. הפעילו פרק של One Pace (צריך שיהיה מותקן התוסף הרשמי של One Pace — ראו למטה).
  4. בחרו בכתוביות בעברית.

המסלול הרגיל עובד בנגן של Stremio. לעיצוב מוקפד יותר, השתמשו ב־mpv.
